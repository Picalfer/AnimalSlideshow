# AnimalSlideshow

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)

Простое приложение, показывающее слайд-шоу изображений животных с возможностью запуска и остановки.

## О проекте

Приложение последовательно переключает изображения через Timer, обновляя ImageView каждые 2 секунды. Кнопка старт/стоп управляет началом и завершением слайд-шоу.

**Основной функционал:**
- Переключение изображений по таймеру  
- Кнопка старт/стоп  
- Пример работы с Timer, TimerTask и runOnUiThread  
- Использование массива ресурсов drawable  

## Технологии

* Kotlin  
* Android Views  
* Timer / TimerTask  
* ImageView / ImageButton
